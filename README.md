# 🚀 WX Launcher

**WX Launcher** - мощный лаунчер для Windows с горячими клавишами и Discord интеграцией.

## 📥 Скачать

[Скачать последнюю версию](https://github.com/waxtoaxx/WX-Launcher/releases/latest)

## ✨ Возможности

- ⚡ Горячие клавиши для запуска приложений
- 🎮 Discord Rich Presence
- 🌙 AMOLED и светлая тема
- 🖼️ Автоматическое извлечение иконок из .exe
- 🚀 Автозапуск с Windows
- 📦 Работает без установки

## 🖥️ Системные требования

- Windows 10 / 11
- Discord (опционально)

## 📦 Установка

1. Скачайте `WX_Launcher.exe` из [релиза](https://github.com/waxtoaxx/WX-Launcher/releases)
2. Запустите файл
3. Добавляйте приложения через кнопку "Обзор"

## 🎮 Discord RPC

Включите Discord статус в настройках, чтобы показывать:
- Запущенные приложения
- Системное время
- Время работы

## 🌙 Темы

- **AMOLED** - темная тема для OLED-экранов
- **White** - светлая тема

## 🔧 Горячие клавиши

Назначайте любые комбинации клавиш для быстрого запуска приложений
